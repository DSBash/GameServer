# GameServer
GameServer with chat and whiteboard : [GameServer](https://github.com/DSBash/GameServer)

## Features

* Sychronise and ASync TCP connections
* Public and Private Chats

## Shared Commands
* /msg <UserName> <Msg>
* /save
* /export
## Host Commands
* /send
## Client Commands
* /picme
